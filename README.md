## Getting Started

These instructions will let you know how to use the library.

### Prerequisites

Things you need to install before using this library.

```shell
node: 10.15.0
nvm: 0.33.6
```

### Installing

sing npm:

```shell
npm i --save selector
```

Using yarn:

```shell
yarn add --dev selector
```

### Unit tests

Using jest:
